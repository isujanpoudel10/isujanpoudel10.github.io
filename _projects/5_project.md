**Timeline:** 2019 - 2022

This project focused on an assistive smart waste container designed for specially abled users. It combined sensing, actuation, RF remote operation, and IoT connectivity to improve accessibility and real-time monitoring.

**Highlights**

- Developed a smart waste container using IR sensors and a servo motor with RF remote operation.
- Designed the system to support specially abled users through easier interaction and automation.
- Enabled IoT-based communication to update a webpage with real-time bin status and system activity.
- Integrated embedded control, sensing, and communication in a complete prototype workflow.

**Focus Areas**

- Assistive technology
- Embedded systems
- RF control
- IoT monitoring
