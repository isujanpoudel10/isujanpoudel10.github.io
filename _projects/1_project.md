**Timeline:** 2025 - Present

This project focuses on semantic image communication over noisy wireless channels using a learning-based end-to-end framework. The goal is to transmit task-relevant visual information efficiently while maintaining perceptual quality under channel impairments.

**Highlights**

- Developed an end-to-end deep learning framework for semantic image transmission over noisy communication channels.
- Trained a generative adversarial network (GAN) to reconstruct high-fidelity images at the receiver.
- Evaluated reconstruction quality using perceptual and distortion-based metrics.
- Studied learning-based communication strategies for efficient representation and transmission relevant to next-generation wireless systems.

**Focus Areas**

- Semantic communications
- Deep learning for communications
- Wireless system optimization
- Robust image reconstruction under channel noise
