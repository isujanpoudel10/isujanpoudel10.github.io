**Timeline:** 2022 - 2023

This project studied the GFSK communication scheme used by Sanosat-1, a CubeSat developed by Orion Space. I replicated the modulation and demodulation pipeline in GNU Radio to analyze digital communication behavior and satellite telemetry.

**Highlights**

- Implemented the GFSK communication scheme of Sanosat-1 using GNU Radio.
- Worked on signal processing, software-based modem development, and telemetry analysis for CubeSat communication.
- Analyzed received satellite data to understand communication parameters and system behavior.
- Applied deep learning methods to detect sudden changes and anomalies in telemetry data.

**Focus Areas**

- Digital communications
- GNU Radio
- Signal processing
- CubeSat telemetry analytics
