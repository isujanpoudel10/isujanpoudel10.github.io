**Timeline:** Spring 2019

This project implemented an automated multilevel room temperature control system using sensing, analog decision logic, and actuator control. It was later extended with IoT-based remote monitoring for real-time system visibility.

**Highlights**

- Designed a temperature control system using an LM35 sensor, comparator circuit, and motor driver.
- Built a multilevel control workflow to respond to changing temperature conditions.
- Extended the system with IoT-based remote monitoring to observe temperature levels and fan states in real time.
- Combined sensing, control, and monitoring in a compact hardware-oriented prototype.

**Focus Areas**

- Control systems
- Embedded design
- Sensor integration
- IoT monitoring
