**Timeline:** 2020 - 2022

This project developed an IoT-based monitoring system for solid waste collection using embedded sensing and wireless data transmission. The system was designed to support efficient collection, centralized monitoring, and data-driven decision making.

**Highlights**

- Developed a monitoring system using ultrasonic sensors and wireless transmission to a central server.
- Integrated sensing, embedded hardware, and communication components for end-to-end implementation and testing.
- Used machine learning methods and frameworks for decision support and collection optimization.
- Explored encoder-decoder based compression strategies for efficient data storage.

**Focus Areas**

- IoT systems
- Embedded sensing
- Wireless monitoring
- Data-driven infrastructure applications
